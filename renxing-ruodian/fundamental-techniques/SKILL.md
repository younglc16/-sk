# 待人基本技巧 · 三大原则

> 源自戴尔·卡耐基《人性的弱点》第一篇

## R（原文精要）

### 原则一：不批评、不责怪、不抱怨

**原文引用：**
> "批评是没有用的，因它使人增加一层防御，而且竭力的替自己辩护。批评也是危险的，它会伤害了一个人的自尊，和自重的感觉，并激起他的反抗。" —— 戴尔·卡耐基

> "一百次中有九十九次，没有人会为了任何一桩事情来批评他自己，无论错误到如何的程度。" —— 戴尔·卡耐基

**关键案例：**
- 双枪克劳雷罪大恶极，但自认为"一颗仁慈的、不愿意伤害任何人的心"
- 林肯写信严厉批评弥特将军，但从未寄出——他学会了"不要评议人，免得为人所评议"

### 原则二：献出真实、诚恳的赞赏

**原文引用：**
> "人类天性至深的本质，就是渴求为人所重视。" —— 威廉·詹姆斯

> "世界上最容易摧毁一个人志向的，那就是上司所给他的批评。我从来不批评任何人，我只给人们工作的激励。我是急于称赞，而迟于寻错。" —— 司华伯

**关键案例：**
- 司华伯年薪百万，不是因为懂钢铁，而是因为善于赞赏和鼓励
- 洛克菲勒在合伙人亏损百万后，赞赏他保全了60%的投资
- 齐格飞给每位表演的歌女加薪送花

### 原则三：引起别人的渴望

**原文引用：**
> "世界上唯一能影响对方的方法，就是谈论他所要的，而且还告诉他，如何才能得到它。" —— 戴尔·卡耐基

> "行动是由我们基本欲望所产生的。对于未来想要说服人家的人，最好的建议，是无论在商业中、家庭中、学校中、政治中，都要先激起对方某种迫切的需要，若能做到这点就可左右逢源，否则到处碰壁。" —— 欧弗斯屈脱教授

**关键案例：**
- 卡耐基用利弊分析表说服饭店经理只涨50%而非300%租金
- 用"打大孩子"激励小孩吃饭；用"自己的床"解决尿床问题

## I（方法论骨架）

### 原则一：不批评法则

```
触发场景 → 你对他人的行为感到不满
   ↓
第一步：暂停 → 不要立即批评或指责
   ↓
第二步：理解 → 问自己"他为什么要这样做？"
   ↓
第三步：共情 → "如果我在他的处境，我也会..."
   ↓
第四步：选择 → 要么不处理，要么用其他方式（赞赏/引导）
```

### 原则二：真诚赞赏法则

```
触发场景 → 你发现某人做对了某事
   ↓
第一步：观察 → 发现具体可赞赏的行为（越具体越好）
   ↓
第二步：真诚 → 确认你的赞赏发自内心
   ↓
第三步：表达 → 当面说出赞赏+具体原因
   ↓
第四步：避免谄媚 → 保持客观，不夸张
```

### 原则三：激发渴望法则

```
触发场景 → 你想让别人做某事
   ↓
第一步：转换视角 → 停止思考"我想要什么"
   ↓
第二步：调研 → 了解对方真正想要的是什么
   ↓
第三步：连接 → "我如何能让他想要做这件事？"
   ↓
第四步：表达 → 从对方的需求出发说明利益
   ↓
第五步：验证 → 观察对方反应，调整策略
```

## A1（书中案例应用）

**不批评原则的应用：**
林肯在年轻时会写匿名信讥笑政客，几乎导致决斗。此后他终生不再批评任何人。内战期间，弥特将军违背命令让李将军逃脱，林肯写了严厉的信却未寄出，因为他意识到批评只会让弥特辩护、辞职，而不是改正。

**真诚赞赏原则的应用：**
司华伯管理钢铁厂时，从不批评工人。他用赞赏替代批评，激发工人的热诚。他说："我还没有找到一个人，无论他如何伟大，不是在被赞许的情形下，比在被批评的情形下，更能够成就伟大的事业。"

**激发渴望原则的应用：**
卡耐基为降低舞厅租金，没有直接要求降价，而是画了一张"利弊分析表"，站在饭店经理的立场分析：涨价会让饭店失去有价值的广告客户，不涨价能获得高端客群。结果只涨了50%而非300%。

## A2（未来触发场景）

**调用此skill的信号词：**

| 触发场景 | 应调用原则 |
|---------|-----------|
| 你想批评下属/孩子/伴侣 | 原则一：不批评——换为理解 |
| 你发现别人做对了，但你觉得理所当然 | 原则二：真诚赞赏——立即表达 |
| 你需要别人配合你做某事 | 原则三：激发渴望——从对方需求出发 |
| 沟通中对方变得防御/对抗 | 原则一+三：停止批评，转到对方需求 |
| 你想激励团队/家人做得更好 | 原则二：赞赏已做好的部分 |

## E（执行步骤）

### 场景1：避免批评（代替方案）

1. 当你想批评时，先做3次深呼吸
2. 问自己：批评能改变什么？还是会让他更防御？
3. 换成问问题："你觉得这个情况怎么样？"
4. 或描述事实而不评价："报告中有三处数据不一致"
5. 合作寻找解决方案："我们怎么改进？"

### 场景2：给出真诚赞赏

1. 每天注意3件别人做对的小事
2. 用"具体行为+正面影响"格式表达：
   - 错误："你很棒"
   - 正确："你昨天熬夜完成方案，让团队按时提交了标书"
3. 当面说，或写便条/邮件
4. 每周至少对家人、同事各表达一次赞赏

### 场景3：激发他人行动

1. 问自己："他想要什么？"（金钱/认可/省事/安全感/成长）
2. 设计沟通话术时，把对方利益放在前面
3. 用"因为...所以..."结构连接对方需求和你的请求
4. 让对方看到清晰的利益路径
5. 必要时用纸笔画出双方利益对照表

## B（边界条件）

| 原则 | 不适用场景 | 注意事项 |
|------|-----------|---------|
| 不批评 | 需立即制止的危险行为（如儿童碰火） | 对严重错误仍需指出，但用温和方式 |
| 真诚赞赏 | 对方做了坏事或伤害他人 | 赞赏必须具体，空泛赞赏 = 谄媚 |
| 激发渴望 | 对方的需求与你无法调和 | 不能操纵——必须给对方真实利益 |

### 卡耐基本人的盲点
- 本书所有原则假设对方是理性的、善意的，不适用于恶意/病态人格
- 文化差异：东亚高语境文化中，"不批评"可能需要更委婉的表达
- 过度使用"激发渴望"可能让对方感觉被操控——真诚是根本前提
