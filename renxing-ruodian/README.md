# 《人性的弱点》— 人际关系完整技能体系

> 📖 戴尔·卡耐基 (Dale Carnegie) 经典著作 · 1936 年出版  
> 🔧 使用 [book2skill](https://github.com/anthropics) 蒸馏为可执行 AI Skill  
> 📅 蒸馏日期：2026-07-02

## 📋 项目简介

将卡耐基《人性的弱点》全书的方法论蒸馏为一套**完整的人际关系技能体系**，总计 **37 条原则与方法**，覆盖待人、社交、说服、领导、家庭五大场景。

AI 可根据问题场景自动匹配对应的功能模块，并在回答末尾标注使用了哪些模块（如 `[M1+M3]`）。

## 🏗️ 体系架构

本 Skill 包含五个功能模块，自下而上构成人际关系金字塔：

```
              ┌──────────────┐
              │  M5: 家庭和睦 │  ← 亲密关系层（7 方法）
              ├──────────────┤
              │  M4: 领导他人 │  ← 管理/批评层（9 方法）
              ├──────────────┤
              │  M3: 得人同意 │  ← 商业/说服层（12 方法）
              ├──────────────┤
              │  M2: 赢得喜欢 │  ← 社交/日常层（6 方法）
              ├──────────────┤
              │  M1: 待人基本 │  ← 底层心态层（3 原则）
              └──────────────┘
```

### 底层三定律（贯穿所有模块）

| 定律 | 含义 |
|------|------|
| **自重感定律** | 人类最深切的冲动是"成为重要人物的欲望" |
| **条件反射定律** | 批评→防御，赞赏→合作 |
| **自我中心定律** | 人们首先关心自己——要影响他人须从对方需求出发 |

## 📦 模块说明

### M1 · 待人基本技巧（3 原则）
| # | 原则 | 一句话 |
|---|------|-------|
| 1 | 不批评、不责怪、不抱怨 | 批评只会让人防御，从理解开始 |
| 2 | 献出真诚的赞赏 | 人类渴求被重视——赞赏是最好的激励 |
| 3 | 引起别人的渴望 | 从"他想要什么"出发，而不是"我想要什么" |

### M2 · 赢得他人喜欢（6 方法）
真诚关心 → 微笑 → 记住名字 → 善于倾听 → 谈论对方兴趣 → 让对方感到重要

### M3 · 得人同意（12 方法）
**四阶段流程：** 建立安全氛围 → 引导认同方向 → 深度理解对方 → 激发行动意愿

包括：避免争论、尊重意见、坦诚认错、友善开场、苏格拉底"是"法、让对方多说话、让他觉得是自己的主意、换位思考、同情对方、激发高尚动机、戏剧化表达、提出挑战

### M4 · 领导他人（9 方法）
**批评三明治模型：** 先赞赏 → 间接指出/自省/提问 → 保全面子 → 赋予好名声/鼓励进步

### M5 · 家庭和睦（7 方法）
**三层次：** 停止破坏（不唠叨/不改造/不批评）→ 积极滋养（赞美/细节/礼貌）→ 认知升级（性知识）

## 🚀 使用方式

### 作为 AI System Prompt

直接将 `SKILL.md` 的全部内容作为 System Prompt 使用。AI 会自动根据用户问题匹配对应模块，并在回答末尾标注 `[M1]`~`[M5]` 的模块代号。

### 场景速查

| 你在什么场景 | 调用哪个模块 | 核心方法 |
|-------------|------------|---------|
| 想批评下属/孩子/伴侣 | M1 → 原则一 | 不批评，改理解 |
| 需要激励团队 | M1 → 原则二 + M4 | 赞赏进步 + 赋予好名声 |
| 参加社交聚会 | M2 | 微笑 + 关心 + 倾听 + 记住名字 |
| 要说服客户/老板 | M3 | 避免争论 → 换位思考 → 从他的利益出发 |
| 要给同事反馈 | M4 | 批评三明治 |
| 和伴侣吵架了 | M5 | 停听想说 + 不批评 + 保持礼貌 |
| 想和某人建立关系 | M2 → 方法1+5+6 | 关心 → 谈他兴趣 → 让他感到重要 |

## ⚠️ 诚实边界

1. **善意假设**：本体系假设对方理性善意，不适用于恶意/病态人格
2. **文化差异**：基于 20 世纪美国文化，东亚高语境文化中需调整表达
3. **真诚是前提**：所有技巧若失去真诚就是操控——"诚于嘉许，宽于称道"
4. **不可替代专业帮助**：严重婚姻危机、心理创伤等需专业咨询
