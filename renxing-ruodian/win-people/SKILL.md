# 赢得他人喜欢 · 六种方法

> 源自戴尔·卡耐基《人性的弱点》第二篇

## R（原文精要）

### 方法一：真诚地对别人发生兴趣

**原文引用：**
> "一个不关心别人，对别人不感兴趣的人，他的生活必遭受重大的阻碍、困难，同时会替别人带来极大的损害、困扰，所有人类的失败，都是由于这些人而才发生的。" —— 阿得洛（维也纳著名心理学家）

> "如果我们只是想使人注意，使人对我们发生兴趣，我们永远不会有真诚的朋友。"

**关键案例：**
- 罗斯福总统记住白宫每位仆人的名字，亲切问候
- 塞斯顿魔术师上台前对自己说："我爱我的观众"
- 银行职员华特用邮票（经理儿子收集）打开对方心扉

### 方法二：微笑

**原文引用：**
> "一个人的行动，比他所说的话，更有具体的表现，而人们脸上的微笑，就有这样的表示：『我喜欢你，你使我快乐，我非常高兴见到你！』"

**关键案例：**
- 司华伯的微笑被认为"值一百万元"
- 股票经纪人司丁哈丹用微笑改变婚姻和事业
- 中国古语："不笑莫开店"

### 方法三：记住对方的名字

**原文引用：**
> "一般人对自己的姓名，比把世界上所有的姓名堆在一起的总数，还感到重要，和关心。把一个人的姓名记住，很自然的叫出口来，你已对他含有微妙的恭维。"

**关键案例：**
- 吉姆能叫出5万人的名字
- 卡耐基用对方名字命名钢厂赢得订单
- 拿破仑三世记住每一个见过的人的名字

### 方法四：做一个善于倾听的人

**原文引用：**
> "很少人能拒受那专心注意所包含的谄媚。" —— 伍福特

**关键案例：**
- 卡耐基与植物学家谈话：自己几乎没说话，却被称赞"最风趣、优美谈吐"
- 百货公司负责人用倾听解决顾客投诉
- 电话公司调解员倾听3小时化解争端

### 方法五：谈论对方感兴趣的事

**原文引用：**
> "深入人们心底的最佳途径，就是对那人讲他知道得最多的事物。"

**关键案例：**
- 罗斯福与任何访客都能聊得投机，事先研究对方感兴趣的话题

### 方法六：让对方感到自己重要

**原文引用：**
> "人类天性至深的本质，就是渴求为人所重视。"

**关键案例：**
- 爱默生："凡我所遇到的人，都有胜过我的地方，我就学他那些好的地方"

## I（方法论骨架）

```
人际吸引力金字塔（从底层到顶层）：

第六层：让对方感到重要 ─── 最高级别的尊重
第五层：谈论对方兴趣 ───── 建立深度连接
第四层：善于倾听 ───────── 让对方畅所欲言
第三层：记住对方名字 ──── 微妙的赞赏
第二层：微笑 ──────────── 打开第一扇门
第一层：真诚关心 ───────── 一切的基础
```

**核心逻辑：** 每一层都建立在上一层的真诚之上。从关心开始，以让对方感到重要为顶点。

## A1（书中案例应用）

**华特的银行调查案例：**
华特需要从一位经理那里获取业务信息，经理不配合。华特注意到经理在帮儿子收集邮票，于是从银行国外汇兑部拿来邮票。第二次拜访时他先聊邮票和经理儿子的照片，然后经理主动提供了全部信息，还叫来下属补充资料。

**电话公司调解员案例：**
一位顾客怒斥电话公司，调解员登门后什么也不说，只是倾听。前后去了四次，每次听老先生发牢骚数小时。最后老先生不仅付清欠款，还撤销了投诉。

## E（执行步骤）

### 方法一：真诚关心

1. 与人交谈前，告诉自己："我对这个人充满兴趣"
2. 问开放性问题："你是怎么开始做这行的？""你最喜欢什么？"
3. 记住对方提到的关键信息（家人、爱好、最近做的事）
4. 下次见面时主动问候这些信息
5. 建立"关心系统"：记录朋友的生日、重要事件

### 方法二：微笑

1. 出门前照镜子，调整表情
2. 与人目光接触时，主动微笑
3. 打电话时也微笑——声音会传递表情
4. 心情不好时："强迫自己微笑"，行为会改变感受

### 方法三：记住名字

1. 初次见面：确认听得清楚，问"请问怎么拼写？"
2. 重复使用：对话中自然地多叫几次对方名字
3. 联想记忆：将名字与面部特征/场景关联
4. 记录：事后记在手机/笔记本上
5. 建立名字库：定期复习

### 方法四：善于倾听

1. 面对对方，保持眼神接触
2. 不打断——让对方把话说完
3. 给予反馈：点头、"嗯嗯"、"然后呢？"
4. 提问跟进："后来怎么样了？""你是怎么做到的？"
5. 不要在脑中准备你接下来要说什么——专注于听

### 方法五：谈论对方兴趣

1. 提前准备：会议/聚会前了解对方背景
2. 直接提问："你最近在忙什么有趣的事？"
3. 注意对方眼睛发亮的话题——那就是他的兴趣所在
4. 往那个方向深入提问
5. 如果你不懂，诚实但好奇地学习

### 方法六：让对方感到重要

1. 真诚地发现对方的独特之处
2. 用尊重的语气和称呼
3. 请对方帮忙/给建议："这方面你是专家，请教一下..."
4. 公开肯定对方的贡献
5. 最核心原则：**每个人都觉得自己很重要——让他知道你也这样认为**

## B（边界条件）

| 方法 | 不适用场景 | 注意事项 |
|------|-----------|---------|
| 真诚关心 | 对方明确表示不想交流 | 关心不能变成打探隐私 |
| 微笑 | 严肃场合（葬礼、正式投诉） | 微笑必须真诚，否则是"皮笑肉不笑" |
| 记住名字 | 对方名字极其复杂且明确说"叫我小X就好" | 尊重对方偏好 |
| 善于倾听 | 对方在恶意攻击或言语骚扰 | 设置边界，适时打断 |
| 谈论对方兴趣 | 对方兴趣是违法/不道德行为 | 保持价值观一致 |
| 让对方感到重要 | 对方是自恋型人格 | 过度满足会助长其自恋 |
