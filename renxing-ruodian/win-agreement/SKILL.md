# 得人同意 · 十二种方法

> 源自戴尔·卡耐基《人性的弱点》第三篇

## R（原文精要）

### 方法一：避免争论
> "辩论、争论的结果，十次中有九次，会使对方更坚持他自己的见解。" "唯一能从争论中得到最大收获的方法，就是避免争论。"

### 方法二：尊重意见
> "你决不能直接的对一个人说：『你错了。』那等于直接向他的判断力挑战。" 「如果你是对的，你要温和地、巧妙地让对方认同你；如果你是错的，你要迅速坦诚地承认。」

### 方法三：坦诚认错
> "任何主动认错的人，都是在对方没有指责之前，先自己承认——这会让对方想指责你的话全部咽回去，转而为你辩护。"

### 方法四：从友善开始
> "一滴蜜比一加仑胆汁能捉到更多苍蝇。用友善的态度开始，比用敌意更能改变别人的心意。"

### 方法五：让说"是"
> "与别人交谈时，不要先谈你们有分歧的事，而要先强调你们都同意的事。要不断强调你们都追求同一目标，唯一差别是方法而非目的。" —— 苏格拉底的秘密

### 方法六：让对方多说话
> "如果你不同意对方的观点，也不要打断他。让他把话说完——他需要宣泄不满。让他充分表达自己的意见。"

### 方法七：让对方觉得是他自己的主意
> "没有人喜欢被推销，我们更喜欢觉得自己是主动购买的。让对方觉得主意是他自己想的，他会更积极地执行。"

### 方法八：换位思考
> "如果有一个成功秘诀的话，那就是如何得到对方『立场』的能力；由他的观点设想，正同由你自己的观点一样。" —— 亨利·福特

### 方法九：同情对方
> "你明天可能需要请求别人做某事，在这之前先告诉自己：『如果我是他，我会有什么感觉？我能理解他为什么那样做。』这句『如果我是你，我也会这样感受』可以平息最激烈的风暴。"

### 方法十：激发高尚动机
> "每个人都会把自己看得很高尚，并且愿意有一个高尚的动机去做事。要改变一个人的意志，需要激发他高尚的动机。"

### 方法十一：戏剧化表达
> "仅陈述事实是不够的，还需要将事实戏剧化地呈现——用行动、画面、故事，让对方看到你所要表达的。"

### 方法十二：提出挑战
> "每一个成功的人都喜欢挑战、表现自我的机会。『提出挑战』是激励他人最有效的方法之一。"

## I（方法论骨架）

```
说服过程四阶段：

阶段一：建立安全氛围 ───── 方法1-4（避免争论→尊重→认错→友善）
阶段二：引导认同方向 ───── 方法5-7（说是→多听→他的主意）
阶段三：深度理解对方 ───── 方法8-9（换位思考→同情）
阶段四：激发行动意愿 ───── 方法10-12（高尚动机→戏剧化→挑战）

自检：如果对方不同意，退回上一阶段
```

## A1（书中案例应用）

**卡耐基降租金：** 没有争论涨价不合理，而是画出利弊分析表，站在经理立场分析。结果租金只涨50%而非300%。

**克纳夫卖煤：** 不再强调自己的煤好，而是请对方帮忙提供连锁百货的资料。对方感动之余主动提出要买煤。

**卡耐基与布尔姆：** 在竞争太平洋铁路业务时，卡耐基提议合并，并以"布尔姆皇宫汽车公司"命名，布尔姆欣然同意。

**苏格拉底法：** 通过连续问对方只能回答"是"的问题，建立同意惯性，再抛出核心主张。

## A2（未来触发场景）

| 触发场景 | 优先使用的方法 |
|---------|--------------|
| 对方在争论/抬杠 | 方法一（避免争论）+ 方法二（尊重意见） |
| 你发现自己错了 | 方法三（主动认错） |
| 对方情绪激动/敌对 | 方法四（友善）+ 方法九（同情） |
| 你想引导对方接受你的方案 | 方法五（说是）+ 方法七（他的主意） |
| 对方不愿说话/防御 | 方法六（让对方多说话） |
| 需要争取利益相关方支持 | 方法八（换位思考）+ 方法十（高尚动机） |
| 对方犹豫不决 | 方法十一（戏剧化）+ 方法十二（挑战） |
| 任何说服场景的开始 | 方法四（从友善开始） |

## E（执行步骤）

### 说服工作流

**Step 1：准备阶段**
1. 明确你的目标——不是"赢"，而是"达成共识"
2. 思考对方的立场和需求（方法八）
3. 找到双方共同的利益点
4. 设计开场—从双方都同意的事情开始（方法五）

**Step 2：建立氛围**
1. 用友善的态度和语气开场（方法四）
2. 如果对方有不同意见，先说"你说得有道理"（方法二）
3. 如果自己有问题，先承认（方法三）
4. 绝对避免说"你错了"或直接反驳

**Step 3：引导对话**
1. 多提问，少陈述——让对方多说话（方法六）
2. 用问题引导对方自己得出结论（方法七）
3. 对方说话时表示理解和同情（方法九）
4. 适时用"你是说……吗？"确认理解

**Step 4：推动行动**
1. 将方案与高尚动机连接（方法十）
2. 用故事/比喻/数据视觉化你的想法（方法十一）
3. 为对方设定一个可完成的挑战目标（方法十二）

## B（边界条件）

| 方法 | 不适用场景 | 注意事项 |
|------|-----------|---------|
| 避免争论 | 在伦理/法律底线问题上 | 原则问题不可回避 |
| 尊重意见 | 对方意见涉及歧视/伤害他人 | 合理表达不认同 |
| 主动认错 | 事实不在你错时 | 不要为讨好而过度认错 |
| 苏格拉底法 | 对方对你的意图已有防备 | 需要真诚而非技巧感 |
| 激发高尚动机 | 对方明确表示"我就是自私" | 尊重对方价值观 |
| 戏剧化表达 | 保守/严肃的正式场合 | 适度使用 |

### 整体局限
- 本书原则建立在"真诚"前提上——用技巧替代真诚必然失败
- 这些方法更适用于平等关系，权力不对等时需要调整
- 文化差异：在直接沟通文化（如荷兰、德国）中可能显得绕弯子
